## How to Publish on GitHub Pages

### Step 1: Create a GitHub Repository
1. Go to [github.com](https://github.com) and log in
2. Click the **+** icon → **New repository**
3. Name it `smithrentalevents` (for username.github.io/smithrentalevents)
4. Make it **Public**
5. Click **Create repository**

### Step 2: Upload Files
Option A - Drag and drop:
1. In your new repo, click **uploading an existing file**
2. Drag all files and the `assets` folder
3. Click **Commit changes**

Option B - Using Git:
```bash
cd "G:\Mi unidad\smith events"
git init
git add .
git commit -m "Initial website upload"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/smithrentalevents.git
git push -u origin main
```

### Step 3: Enable GitHub Pages
1. Go to your repository **Settings**
2. Click **Pages** in the left sidebar
3. Under **Source**, select **Deploy from a branch**
4. Select **main** branch and **/(root)** folder
5. Click **Save**
6. Wait 2-3 minutes for the site to deploy

### Step 4: Your URL
The site will be available at:
```
https://anibru300.github.io/smithrentalevents/
```

If you want a custom domain, add a `CNAME` file with your domain name.
